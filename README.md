# Claude-to-Gemini and Gemini-to-Claude Bidirectional Session Migration Tool

This tool facilitates the seamless migration of sessions between the Claude and Gemini environments. It provides users with the ability to transfer ongoing conversations and data from Claude to Gemini and vice versa, ensuring a smooth transition without losing context or information.

## Features
- Supports bidirectional session migration
- Maintains context during migration
- Easy-to-use interface

## Usage
1. Install the tool.
2. Follow the instructions to connect to your Claude and Gemini environments.
3. Choose the migration direction (Claude to Gemini or Gemini to Claude).
4. Initiate the session transfer.
